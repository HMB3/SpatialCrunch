About

SpatialCrunch provides spatial data analysis services, focusing on public interest projects.

We are an interdisciplinary team of spatial analysts, geographers and researchers with over 10 years’ experience in analysing, visualizing and communicating the varying distributions of complex data in space and time. We have developed generalized skills in building pipelines for geospatial analysis with feature and raster data.

For example, open-source functional programming skills can be applied to contexts where bespoke software has not yet been written, and is too expensive. The processing and synthesis of raster and feature datasets can be automated to find spatial and statistical patterns between variables, visualize output and document both processes and results for different audiences in different contexts.
